퀴즈 추가 방법

1. 이미지를 GitHub anime 폴더에 업로드
2. [퀴즈 추가] 탭에서 파일명, 정답, 힌트 입력 → 추가
3. [내보내기] → JSON 복사 → GitHub quiz.json 업데이트
4. [내보내기] → HTML 생성 → GitHub quiz/anime/[번호].html 파일 생성
5. GitHub Desktop Commit → Push

같은 만화 여러 장 추가

같은 정답이라도 파일명(이미지)이 다르면 별개의 퀴즈로 추가할 수 있어요.
예) 원피스_1.png, 원피스_2.png
